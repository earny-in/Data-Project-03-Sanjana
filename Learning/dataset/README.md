# 📰 Fake News Detection Dataset

## 📌 Overview

This dataset is designed for **Fake News Detection** using Machine Learning and Natural Language Processing (NLP). It contains news articles along with their metadata and labels indicating whether the news is **Real** or **Fake**.

The dataset can be used for:
- Fake News Detection
- Text Classification
- NLP Projects
- Data Analysis
- Machine Learning Practice

---

# 📊 Dataset Information

- **Total Records:** 20,000
- **Total Features:** 7
- **Target Column:** `label`

---

# 📂 Dataset Columns

| Column | Description |
|---------|-------------|
| title | Title of the news article |
| text | Complete news content |
| date | Publication date of the article |
| source | Source or publisher of the news |
| author | Author of the article |
| category | News category (Politics, Sports, Technology, etc.) |
| label | News type (Real or Fake) |

---

# 🎯 Objective

The main objective of this dataset is to build a Machine Learning model that can classify whether a news article is **Real** or **Fake**.

# 🧹 Data Preprocessing

Before training the model, the following preprocessing steps are recommended:

- Remove missing values
- Convert text to lowercase
- Remove punctuation
- Remove stopwords
- Tokenization
- Stemming/Lemmatization
- TF-IDF Vectorization
